# Smart City Web Portal

This project is a Smart City Web Portal built using the MERN stack.

## Features
- User authentication
- City service information
- Complaint registration
- Smart city updates

## Technologies Used
- MongoDB
- Express.js
- React.js
- Node.js

## Project Structure
Frontend – React application
Backend – Node.js and Express server

## How to Run

Frontend
cd frontend
npm install
npm start

Backend
cd backend
npm install
node server.js
